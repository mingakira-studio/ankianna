# Research: 互动角色设计
日期: 2026-03-16
深度: 轻量

## 任务上下文
升级现有 DinoCharacter（简单 SwiftUI Shape 绘制恐龙）为更精细的 3D 卡通龙角色。

## Research 发现
- DinoCharacter 当前: Circle/Ellipse/Capsule 组合, 纯色填充, 5个状态(idle/thinking/happy/encourage/celebrate)
- 每个状态只有固定一句台词
- 动画: 仅 bouncing scale 效果
- MascotView API: `MascotView(state: MascotState)`, 被 LearningView/practiceView 使用
- 已有 reducedMotion 支持
- DesignTokens 设计系统可复用

## 关联文件
- 修改: Views/Learning/MascotView.swift — 重写 DinoCharacter → DragonCharacter

## 风险与边界
- 纯 SwiftUI 绘制复杂度上限, 不引入 3D 框架
- 保持 MascotView(state:) 外部 API 不变
- 保持 reducedMotion 支持

## 技术方案
- 目标: 将简单恐龙升级为精细3D质感卡通龙, 丰富表情和台词
- 方案: 渐变填充+高光阴影层伪3D, 增加龙角/翅膀/腹部/龙须, 眉毛/星星眼/眼泪表情, 每状态多条随机台词
- 边界: 不用SceneKit/RealityKit, 不用外部图片, 保持纯SwiftUI
- 交付层级: MascotView.swift单文件重写
- 验收标准: 龙角色有明显3D质感(渐变+阴影), 每次回答台词不重复
