## 技术方案
- 目标: 每次书写产生细粒度数据点, 自动同步到 Mac 用于事后分析
- 方案: WritingAttempt SwiftData模型 + 各ViewModel埋点 + 练习完成写JSON到iCloud Documents自动同步
- 边界: 不做HTTP API, 不做实时查询, 不做认证
- 交付层级: 数据层(模型+埋点) + 同步层(iCloud Documents)
- 验收标准: iPad练习后, Mac ~/Library/Mobile Documents/ 下自动出现learning_data.json

## 风险与边界
- iCloud 同步延迟不可控（通常几秒到几分钟）
- 游戏模式埋点需要传入 modelContext（当前 ViewModel 没有）
- 需要开启 iCloud Documents capability
